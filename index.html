Меня зовут Сергей

Доброго времени суток!

Меня зовут Скоромник Сергей Николаевич. Я захотел стать FrontEnd developer'ом и поэтому появилось это портфолио.

Я закончил среднюю школу в 2006 году, после чего обучался в институте по специальности правоведение.

Затем я работал по специальности сразу после окончания учебного заведения, начиная с 2011 года.

В 2019 году впервые попробовал себя в верстке на бесплатных курсах от RSSchool. Обучение проходил с переменным успехом, однако не сумел пройти его до логического финала по различным причинам. После чего успешно отложил мысли о FrontEnd.

Однако в сентябре 2021 года я решился начать обучаться самостоятельно, с желанием разобраться и желанием понимать, как все работает. С тех пор я окончил различные курсы и практики, указанные ниже.

По окончании курсовя занимался практикой различной степени сложности. В данном портфолио присутствуют 2 макета для наглядной демонстрации было/стало.

Прошел подготовительный курс для дальнейшего зачисления на курс по Java Script на платформе RSSchool.

В настоящее время прохожу курсы по изучению REACT и TypeScript.

Касаемо английского языка, то он у меня находится на уровне А2-В1, не буду лукавить без языковой практики стал забываться.

Также имею водительское удостоверение с правом управления категории Б.

О себе могу добавить: играю в командные виды спорта, любимый вид спорта волейбол (играю в команде любительсой лиги), люблю походы и отдых вне помещений, играю на гитаре.

Что я использую в работе

HTML5

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта.

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое.

Jquery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует.

Bootstrap

Эта библиотека помогает в создании быстрых, отзывчивых сайтов.

Figma

Это кросс-платформенный онлайн-сервис для дизайнеров интерфейсов и веб-разработчиков. Разработка интерфейсов происходит в онлайн-приложении.

Git

Это система управления версиями, предназначенная для быстрой и эффективной обработки всего, от небольших до очень крупных проектов.

Gulp

Набор инструментов для автоматизации и улучшения рабочего процесса.

Webpack

Один из самых мощных и гибких инструментов для сборки frontend.

Node.js

Эта платформа позволяет создавать бэкенд для вашего продукта - “мозги”, которые будут выполнять действия, которые пользователь не видит.
